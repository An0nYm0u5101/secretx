# SecretX
Extracting api keys and secrets by requesting each url at the your list.

![](example.png)

### Installation
```
python3 -m pip install -r requirements.txt
```

### Usage
```
python3 secretx.py --list urlList.txt --threads 15

optional arguments: --help  --colorless
```
